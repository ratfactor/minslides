Hello world!

This is minslides, the minimalistic HTML slide presentation "framework".

Press the J key on your keyboard to go the next slide.

minslides is:

  • Almost nothing
  • Just a single web page
  • Keyboard driven slides!

(Try H to go back, then J to return.)

Content is HTML

<div class="slide">Your content here...</div>

Like I said, it's just a web page.

(Now try scrolling up and down like you would on any normal web page.)

The End
